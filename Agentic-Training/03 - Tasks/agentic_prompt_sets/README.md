# Agentic Coding Demo — Prompt Sets

1. `01-security-audit.md` — security audit and bounded remediation
2. `02-antipatterns.md` — antipattern detection and bounded refactoring
3. `03-performance.md` — performance analysis and measured optimization
4. `04-dotnet-framework-upgrade.md` — conservative .NET Framework modernization
5. `05-final-integration-review.md` — integrated regression review, corrections and final acceptance

All workflows use only the skills currently present in the repository:

- `explore-repository`
- `implement-task`
- `review-change`
- `validate-project`

Recommended execution rule: use a new chat for every phase so that analysis, independent review, implementation and final review remain separated.
